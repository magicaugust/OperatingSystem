目标 : 
```
用Java实现一个软驱的逻辑
主要目的是为了方便我们将我们用汇编语言写好的程序用这个Java实现的软驱逻辑烧写成一个软驱镜像
这样我们就可以任意控制软驱的所有字节的数据
```
原理 : 
```
首先需要了解软驱的整体结构 : 
1. 一个软驱总共有 1 个盘片
2. 一个盘片有两个盘面 , 每个盘面分别有一个磁头用于读写数据
3. 每一个盘面被分为不同的柱面(磁道) , 根据字面意思理解 , 就是被分为不同的同心圆
4. 每一个盘面会被分为 80 个柱面
5. 每一个柱面又会被切分为 18 个扇区
6. 每一个扇区可以存储 512 字节的数据
7. 那么也就是说 一个软驱的容量可以这样来计算 : 
	512 * 18 * 80 * 2 = 1474560 (字节) = 1440 (KB) ≈ 1.5 (MB)
   这也就解释了为什么我们之前要将我们的软驱镜像文件填充为 1440 KB
   这些字节数转换为 16 进制也就是 0x168000 , 也就是之前的 Java 代码中硬编码的那个数字
-------------------
然后我们需要设计一个软驱都需要有什么样的功能 : 
1. 读取指定盘片 , 指定柱面 , 指定扇区的内容
2. 写内容
	在写入内容的时候 , 首先写入 0 号盘面的 0 号磁道的 1 号扇区(扇区由 1 开始编号)
-------------------
我们可以根据上述的分析 , 先构建出一个 软驱类
```
