目标 : 
```
实现操作系统读取指定扇区内容的功能
具体来说就是 : 
	1. 操作系统内核位于软盘的 0 盘面 , 0 磁道 , 1 扇区
	2. 需要访问的数据位于操作系统扇区的下一个扇区 , 也就是 2 扇区
	3. 通过 BIOS中断 调用实现读取 2 扇区的数据(字符串) , 并输出到屏幕
具体如何使用 BIOS中断 , 在项目根目录的 /resource/pdf/BIOS中断大全.pdf 中已经进行了详细的说明 , 可以查阅具体的流程
```
备注 : 
```
需要在上一节课的基础上修改汇编代码
由于烧写系统镜像的Java代码在上节课已经实现 , 因此只需要调用即可
不对 , 因为要在第二个扇区中也写入数据 , 因此 , 需要修改一部分 Java 代码 , 将我们定义的字符串写入第二个扇区
还有上节课写的 Java 代码虽然实现了功能 , 但是还有一个错误没有修复 , 就是 BIOS 终端在对扇区进行编号的时候是从 1 开始的
但是 Java 代码中使用了从 0 开始编号的规则 , 因此 , 为了避免诸多错误 , 暂时先使用老师的 Java 代码(更加精炼简洁)
```
